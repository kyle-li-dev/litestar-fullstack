# 📋 Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] — 2026-04-23

🎉 **Initial Release** — Project bootstrap with full-stack architecture.

### ✨ Features

- **backend**: Litestar application with JWT authentication, structlog logging, and cryptography support
- **frontend**: React SPA powered by Vite with hot-reload development experience
- **auth**: Complete authentication system — password, OAuth, and TOTP/2FA
- **database**: PostgreSQL with Advanced Alchemy ORM, migrations, and connection pooling
- **email**: React email templates compiled to HTML with SMTP delivery via Mailpit (dev)
- **worker**: Background task processing with SAQ + Redis
- **docker**: Production-ready distroless container and development Docker Compose stack
- **docs**: MkDocs Material documentation site with search, dark mode, and Mermaid diagrams

### 📦 Build System

- UV for Python dependency management and virtual environments
- Bun for JavaScript/TypeScript toolchain
- Hatch build backend with wheel packaging
- OpenAPI schema export and TypeScript client generation

### 🔧 CI/CD

- Pre-commit hooks with conventional commits enforcement
- Ruff linting and formatting for Python
- Biome linting for JavaScript/TypeScript
- Codespell for spell checking
- MyPy + Pyright for type checking
- Slotscheck for `__slots__` validation

### 🧪 Testing

- Pytest with xdist parallel execution
- Coverage reporting with 65% minimum threshold
- pytest-databases for PostgreSQL integration tests
